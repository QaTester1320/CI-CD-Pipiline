CI/CD (Continuous Integration / Continuous Deployment) automates the path from your local machine to the real world.

State 1: Local Dev

Prepare your code and push to GitHub.

git add . git commit -m "init" git push origin main

State 2: Build Engine

Setup the environment and compile the app.

npm install npm run build

State 3: Testing

Run unit tests and security scans.

npm test run security-scan

State 4: Production

Package into Docker and Go Live.

docker build -t shop:v1 . deploy --prod
